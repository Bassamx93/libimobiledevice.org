# Open Terminal

Open a command-line terminal on your machine.

# Installation

```shell-session
$ mkdir -p limd-build
$ cd limd-build
$ curl -Ls -o limd-build-macos.sh https://is.gd/limdmacos
$ bash ./limd-build-macos.sh
```
